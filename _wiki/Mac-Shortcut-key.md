| 符号 | 修饰键 |
| :--: | :-- |
| ⌘ | command |
| ⇧ | shift |
| ⌥ | option |
| ⌃ | control |
| ↩ | return |
| ⌫ | delete |
| ⌦ | forward delete |
| ↑ | up arrow |
| ↓ | down arrow |
| ← | left arrow |
| → | fight arrow |
| ⇞ | page up |
| ⇟ | page down |
| ↖︎ | home |
| ↘︎ | end |

| ⏏ | eject |

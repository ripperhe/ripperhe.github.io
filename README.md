# ripperhe.github.io


这很显然是利用 [GitHub Pages](https://pages.github.com/) 联合 [Jekyll](http://jekyll.com.cn/) 搭建的一个静态博客。

很高兴，有自己的博客了。[欢迎来访~](http://ripperhe.com) 🎉

## My idea

* 不写炫耀性的文章
* 不仅仅为获赞而写文章
* 引用文章注明出处
* 多读高质量的源码及文章
* 学会看英文官方文档是一件有必要的事情
* 认清自己的水平
